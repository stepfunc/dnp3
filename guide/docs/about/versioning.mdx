Our library follows the [semantic versioning](https://semver.org/) specification. Refer to the specification for the meaning of the `<major>.<minor>.<patch>`
fields of the version number.

import sitedata from '../../sitedata.json'

<>The latest release (version {sitedata.version}) is available <a href={`${sitedata.github_url}/releases/tag/${sitedata.version}`}>here</a>.</>
