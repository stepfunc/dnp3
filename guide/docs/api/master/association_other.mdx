You can perform other miscellaneous operations on an association, including:

* Performing ad-hoc (i.e., one time) read requests
* Synchronizing the outstation's time
* Performing warm/cold restarts
* Initiating link status requests to an outstation

You can find details about these operations in the language-specific API docs.
