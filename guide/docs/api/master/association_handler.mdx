`AssociationHandler` handles per-association callbacks unrelated to measurement data. It contains a single method (`get_current_time`) to retrieve a UTC timestamp for time synchronization.

Typically, return the system time (synchronized via NTP) in `get_current_time`. You can use a different clock if desiredâ€”for example, a data concentrator receiving time from an upstream DNP3 master. The examples below retrieve the system time.
